"# Интеграция контейнеров системы" 
"" 
"## Взаимодействие:" 
"1. **setup-master** запускается первым и настраивает Redash" 
"2. **data-generator** начинает генерировать данные" 
"3. Данные сохраняются в **database**" 
"4. Redash визуализирует данные из базы" 
"5. **notebooks** используются для углубленного анализа" 
"" 
"## Docker Compose последовательность:" 
"\`\`\`yaml" 
"services:" 
"  setup-master:    # Контейнер 2: настройка" 
"    restart: 'no'  # Запуск один раз" 
"  " 
"  data-generator:  # Контейнер 1: данные" 
"    depends_on:" 
"      - setup-master" 
"\`\`\`"
